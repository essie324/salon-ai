# Salon AI – Competitor Benchmark

## Purpose
Define the competitive landscape and ensure every feature we build is intentionally stronger, smarter, or more elegant than existing salon systems.

---

# Core Competitors

## 1. Fresha
**Strengths**
- Clean, modern UI
- Fast booking flow
- Marketplace exposure
- Strong client-facing experience

**Weaknesses**
- Limited deep client intelligence
- Weak retention automation
- Minimal stylist-level insight

**We Beat Fresha By**
- Intelligent scheduling (gap filling, suggestions)
- Client memory (formula + history)
- AI receptionist decision-making
- Rebooking automation

---

## 2. Meevo
**Strengths**
- Enterprise-level salon/spa features
- Strong reporting and operations
- Staff management depth

**Weaknesses**
- Heavy, slow UX
- Complex setup
- Not intuitive for modern users

**We Beat Meevo By**
- Simpler UX with same power
- Faster workflows
- AI-assisted decisions instead of manual management

---

## 3. Square Appointments
**Strengths**
- Easy setup
- Built-in payments
- Familiar ecosystem

**Weaknesses**
- Not salon-specific
- Weak service intelligence
- Limited customization

**We Beat Square By**
- Salon-native logic (services, stylists, timing)
- Deep client profiles
- Behavioral and retention intelligence
- Smarter booking flows

---

## 4. Vagaro
**Strengths**
- Built specifically for salons/spas
- Memberships, packages, classes
- Strong industry adoption

**Weaknesses**
- Clunky UI in places
- Limited automation intelligence
- No real AI or predictive system

**We Beat Vagaro By**
- Cleaner, more modern UX
- AI-driven booking + intake
- Client memory system
- Predictive rebooking + retention

---

# Our Position

Salon AI is not just a scheduler.

It is:

> An AI-native salon operating system that combines booking, memory, scheduling intelligence, retention, and receptionist automation.

---

# Core Differentiators

## 1. AI Receptionist
- Converts conversation → booking
- Interprets vague requests
- Suggests optimal appointments

## 2. Smart Scheduling Engine
- Detects gaps
- Suggests revenue opportunities
- Optimizes stylist utilization

## 3. Client Memory System
- Formula tracking
- Visit history
- Behavioral insights

## 4. Retention Intelligence
- Rebooking automation
- Due / overdue detection
- Client lifecycle awareness

## 5. Behavior + Pattern Recognition
- Identifies habits
- Predicts booking behavior
- Flags risk (no-shows, cancellations)

---

# Build Philosophy

Every feature must answer:

1. Does this outperform Fresha in UX?
2. Does this outperform Meevo in power?
3. Does this outperform Square in simplicity?
4. Does this outperform Vagaro in salon-specific value?
5. Does this introduce intelligence (AI, automation, prediction)?

If not, it should not be built or should be redesigned.
